# Natural 1♥

|------|-----------------------------------|
| 1♠!  | NF, usually 8--11, 0--5♠          |
| 1NT! | F, 5+♠ or FG 4+♠                  |
| 2♣   | FG, 4+♣                           |
| 2♦   | FG, 4+♦                           |
| 2♥   | CONST, 3+♥                        |
| 2♠   | PRE, 6+♠                          |
| 2NT! | FG, 4+♥                           |
| 3♣   | INV, 6+♣                          |
| 3♦   | INV, 6+♦                          |
| 3♥   | LIM, 4+♥                          |
